# styled-cssgg
> styled components for [css.gg](https://github.com/astrit/css.gg)

## pre

```bash
npm install styled-cssgg --save
```

## usage

```tsx
import React from 'react'
import { Abstract } from 'styled-cssgg'

export default function() {
  return (
    <div>
      <Abstract />
    </div>
  )
}
```
